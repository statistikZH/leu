## Usage

### Sizing

The icon size can be controlled via the `--leu-icon-size` CSS custom property:

```html
<leu-icon name="check" style="--leu-icon-size: 32px"></leu-icon>
```

### Color

The icon fill uses `currentColor`, so it inherits the text color of its parent. Override it by setting the `color` CSS property:

```html
<leu-icon name="check" style="color: red"></leu-icon>
```
