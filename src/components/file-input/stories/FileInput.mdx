## Usage

The file input is form-associated and supports drag & drop. Selected files are displayed in a list with remove buttons.

### Accepting specific file types

Use the `accept` attribute to restrict which file types can be selected, following the same syntax as the native `<input type="file">`:

```html
<leu-file-input accept="image/*" label="Upload images"></leu-file-input>
```

### Multiple files

Set the `multiple` attribute to allow selecting more than one file at a time.
