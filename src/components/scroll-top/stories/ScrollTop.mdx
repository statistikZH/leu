## Usage

The scroll-to-top button appears automatically when the user scrolls back up after scrolling past the fold. It smoothly scrolls the page to the top when clicked.

Place the element once in your page — it has no configurable properties:

```html
<leu-scroll-top></leu-scroll-top>
```
