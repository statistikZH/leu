## Usage

The checkbox group wraps multiple `<leu-checkbox>` elements and renders them as a fieldset with an optional legend via the `label` attribute.

### Orientation

Set `orientation` to `"vertical"` or `"horizontal"` (default) to control the layout direction.

### Reading the value

The `value` getter returns an array of the checked checkbox values.
