## Usage

The checkbox is form-associated and works with native `<form>` elements.

### Checked vs. defaultChecked

Use `checked` as a JavaScript property to programmatically set the state. Use the `checked` HTML attribute (maps to `defaultChecked`) for the initial state in markup, similar to how native checkboxes work.

### Label

The label is provided via the default slot:

```html
<leu-checkbox value="agree" name="terms">I agree to the terms</leu-checkbox>
```
