## Usage

The input component is form-associated and works with native `<form>` elements. It supports various input types, validation, and visual customization.

### Prefix and suffix

Use the `prefix` and `suffix` attributes to display units or additional labels inside the input field.

### Icon

Set the `icon` attribute to display an icon inside the input field.

### Clearable

Set the `clearable` attribute to show a clear button when the input has a value.

### Validation

The input supports native constraint validation via `pattern`, `required`, `min`, `max`, `minlength`, and `maxlength`. Set `novalidate` to disable validation feedback. Use the `error` attribute to display a custom error message.
