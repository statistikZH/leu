## Usage

The dialog uses the native `<dialog>` element with `showModal()` under the hood, providing built-in backdrop and focus trapping.

### Opening and closing

Use the `show()` and `close()` methods to control the dialog programmatically:

```js
const dialog = document.querySelector("leu-dialog")
dialog.show()
dialog.close()
```

### Action bar

Use the `actionbar` slot to place buttons at the bottom of the dialog:

```html
<leu-dialog label="Confirm">
  <p>Are you sure?</p>
  <leu-button slot="actionbar" variant="secondary">Cancel</leu-button>
  <leu-button slot="actionbar">Confirm</leu-button>
</leu-dialog>
```
