## Usage

### Variants

The tag supports three variants: `"solid"` (default), `"outline"`, and `"ghost"`.

### Custom accent color

Override the tag's accent color with the `--leu-tag-accent-color` CSS custom property:

```html
<leu-tag style="--leu-tag-accent-color: #b01657">Custom color</leu-tag>
```

### Icon

Use the `icon` slot to display a `<leu-icon>` next to the label:

```html
<leu-tag>
  <leu-icon slot="icon" name="confirm"></leu-icon>
  Published
</leu-tag>
```
